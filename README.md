# Konservy-app

Приложение для управления семейными финансами и безопасностью банковских счетов.

## Описание

Konservy-app — это современное мобильное приложение на Kotlin с Material Design 3, которое позволяет:

- 🏦 **Управлять счетами всей семьи** — добавьте счета детей, супругов, родителей
- 🔔 **Получать уведомления о подозрительных операциях** — переводы, крупные снятия, платежи
- 👨‍👩‍👧‍👦 **Контролировать младшее и старшее поколение** — настройте лимиты и уровни доверия
- 📱 **Мониторинг звонков и мессенджеров** — предупреждения о длительных разговорах с незнакомцами
- 🔐 **Двухфакторная аутентификация** — пароль + SMS для максимальной защиты
- 🔒 **Шифрование данных** — все данные защищены современными алгоритмами
- 👥 **Доверенные контакты** — уведомления отправляются всем членам семьи одновременно

## Ключевые функции

### Безопасность
- Двухфакторная аутентификация (пароль + SMS)
- Настройка лимитов на переводы и снятия
- Мониторинг длительности звонков
- Контроль активности в мессенджерах
- Автоматическая блокировка подозрительных операций

### Управление семьёй
- Добавление членов семьи с разными уровнями доверия
- Просмотр счетов всех членов семьи
- Гибкие настройки уведомлений
- Статистика и мониторинг активности

### Интерфейс
- Material Design 3
- Поддержка светлой и тёмной темы
- Интуитивно понятный интерфейс
- Видео-инструкции и помощь

- Kotlin 1.9+

## Инструкция по использованию

1. **Первый вход**: Введите пароль (минимум 4 символа), затем код из SMS (тестовый код: 123456)
2. **Главный экран**: Просмотр баланса, карт и истории операций
3. **Семья и безопасность**: 
   - Добавьте членов семьи через кнопку "+"
   - Настройте уровни доверия (Полный/Стандарт/Ограниченный)
   - Установите лимиты в настройках безопасности
4. **Подозрительные операции**: Получайте уведомления и подтверждайте/блокируйте операции
5. **Помощь**: Видео-инструкции и статьи по использованию

## Функционал макета

1. **Верхняя панель (TopAppBar)**

   - Платёж
   - Ещё

5. **История транзакций**
   - Список последних операций
   - Иконки категорий
   - Цветовая индикация доходов/расходов

6. **Нижняя навигация (NavigationBar)**
   - Главная
   - Платежи
   - Переводы
   - Профиль
