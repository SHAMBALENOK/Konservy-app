# Konservy-app

Современное мобильное приложение для управления семейными финансами и безопасностью банковских счетов.

## 📱 Описание

Konservy-app — это приложение на Kotlin с Material Design 3, которое позволяет:

- 🏦 **Управлять счетами всей семьи** — добавьте счета детей, супругов, родителей
- 🔔 **Получать уведомления о подозрительных операциях** — переводы, крупные снятия, платежи
- 👨‍👩‍👧‍👦 **Контролировать младшее и старшее поколение** — настройте лимиты и уровни доверия
- 📱 **Мониторинг звонков и мессенджеров** — предупреждения о длительных разговорах с незнакомцами
- 🔐 **Безопасная аутентификация** — регистрация, вход по паролю и PIN-код для быстрого доступа
- 🔒 **Шифрование данных** — все данные защищены современными алгоритмами
- 👥 **Доверенные контакты** — уведомления отправляются всем членам семьи одновременно

## 🔐 Процесс аутентификации

### Первый запуск (Регистрация)
1. Ввод данных: email, имя, фамилия, телефон, username, пароль
2. Отправка данных на сервер
3. Установка 4-значного PIN-кода для быстрого входа на этом устройстве

### Вход с другого устройства
1. Ввод username и пароля
2. Доступ к приложению

### Повторный вход на знакомом устройстве
1. Ввод 4-значного PIN-кода
2. Мгновенный доступ к приложению

## 📖 Инструкция по использованию

### Регистрация нового пользователя
1. Нажмите "Зарегистрироваться" на экране входа
2. Заполните все поля: email, имя, фамилия, телефон, username, пароль
3. Подтвердите пароль
4. Нажмите "Зарегистрироваться" — данные будут отправлены на сервер
5. Придумайте и введите 4-значный PIN-код
6. Подтвердите PIN-код повторным вводом
7. Готово! Вы попали в приложение

### Вход существующего пользователя
1. Введите username и пароль
2. Нажмите "Войти"
3. Вы попали в приложение

### Быстрый вход по PIN-коду
1. При повторном открытии приложения на этом же устройстве
2. Введите ваш 4-значный PIN-код
3. Мгновенный доступ к приложению

### Работа с приложением
1. **Главный экран**: Просмотр баланса, карт и истории операций
2. **Семья и безопасность**: 
   - Добавьте членов семьи через кнопку "+"
   - Настройте уровни доверия (Полный/Стандарт/Ограниченный)
   - Установите лимиты в настройках безопасности
3. **Подозрительные операции**: Получайте уведомления и подтверждайте/блокируйте операции
4. **Помощь**: Видео-инструкции и статьи по использованию
