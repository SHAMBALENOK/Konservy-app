# BankApp - Макет банковского приложения на Kotlin с Material Design 3

## Описание
Это макет банковского мобильного приложения, разработанный с использованием:
- **Kotlin**
- **Jetpack Compose** для современного декларативного UI
- **Material Design 3 (MD3)** для актуального дизайна

## Структура проекта

```
BankApp/
├── app/
│   ├── src/main/
│   │   ├── java/com/example/bankapp/
│   │   │   ├── MainActivity.kt          # Главная активность
│   │   │   └── ui/
│   │   │       ├── screens/
│   │   │       │   └── HomeScreen.kt    # Экран главной страницы
│   │   │       └── theme/
│   │   │           ├── Color.kt         # Цветовая палитра
│   │   │           ├── Theme.kt         # Тема приложения
│   │   │           └── Type.kt          # Типографика
│   │   ├── res/
│   │   │   ├── drawable/                # Векторные ресурсы

```

## Функционал макета

### Главный экран включает:

1. **Верхняя панель (TopAppBar)**
   - Название приложения "Мой Банк"
   - Кнопки уведомлений и поиска

2. **Приветственный блок**
   - Персональное приветствие
   - Отображение общего баланса

3. **Быстрые действия**
   - Перевести
   - Пополнить
   - Платёж
   - Ещё

4. **Карточки счетов**
   - Основной счёт
   - Сберегательный счёт
   - Кредитная карта
   - Дизайн в стиле реальных банковских карт

5. **История транзакций**
   - Список последних операций
   - Иконки категорий
   - Цветовая индикация доходов/расходов

6. **Нижняя навигация (NavigationBar)**
   - Главная
   - Платежи
   - Переводы
   - Профиль

## Особенности Material Design 3

- ✅ Использование `material3` библиотеки
- ✅ Цветовые схемы (ColorScheme)
- ✅ Компоненты MD3: TopAppBar, NavigationBar, Surface, Card
- ✅ Система типографики MD3
- ✅ Поддержка темной темы
- ✅ Закругленные углы и тени
- ✅ Адаптивный дизайн

## Требования

- Android Studio Hedgehog (2023.1.1) или новее
- Kotlin 1.9.20+
- Min SDK: 24 (Android 7.0)
- Target SDK: 34 (Android 14)

## Установка и запуск

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Запустите на эмуляторе или реальном устройстве

## Зависимости

Основные зависимости указаны в `app/build.gradle.kts`:
- `androidx.compose.material3:material3`
- `androidx.navigation:navigation-compose`
- `androidx.activity:activity-compose`

## Автор

Макет создан для демонстрации возможностей Jetpack Compose и Material Design 3.
